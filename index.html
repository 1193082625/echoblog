MarkDown

标题

# 标题1
## 标题2
### 标题3

引用

在每行或者每段的最前面加>
> This is a blockquote

列表

使用 * 、 + 或是 - 作为列表标记
有序列表则使用数字接着一个英文句点
如果列表项目用空行分开，在输出HTML时Markdown就会将项目内容用<p>标签包起来

分割线

在一行中用三个以上的 * 、 - 、 _ 来建立一个分割线，行内不能有其他元素，也可以在 * 或 - 中间插入空格

链接

[链接 文字]: http://baidu.com ‘title内容’

强调

被 * 或 _ 包围的字词会被转成用<em>标签包围，用两个 * 或 _ 包起来的话，则会被转成<strong>

图片

![图片的替代文字](图片地址)
到目前为止，Markdown还没有办法指定图片的宽高
