构建工具的作用：

  • 语法检查

  • 文件压缩
    等

安装

使用npm安装需要先安装nodejs，可以直接在nodejs官网下载安装。

1、全局安装gulp，用来在命令行中使用gulp

2、在项目中安装gulp

gulp API

  • gulp.task() //定义任务
  • gulp.src() //读文件
  • gulp.dest() //写文件
  • gulp.watch() //监听
  • gulp.pipe() //管道流
  • gulp插件

在Gulp中，使用的是以Nodejs中的stream(流)为媒介的，它不需要频繁的生成临时文件,首先通过gulp.src()获取到需要的stream，但要注意这个流里的内容不是原始的文件流，而是一个虚拟文件对象流(Vinyl files)，这个虚拟文件对象中存储着原始文件的路径、文件名、内容等信息，然后可以通过stream的pipe()方法把流导入到你想要的地方，比如Gulp的插件中，经过插件处理后的流又可以继续导入到其他插件中，当然也可以通过gulp.dest()把流写入到文件中。

运行gulp

打开命令行工具，并切换到项目目录下，在命令行中执行gulp taskName执行相应任务。如果没有指定任务名，则会执行默认的default任务
