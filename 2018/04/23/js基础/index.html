预编译

预编译发生在函数执行的前一刻
1、创建AO(activation objec)活跃对象，即执行期上下文
2、将形参和变量声明作为AO属性名，值为undefined
3、将实参和形参统一
4、找函数声明，值为函数体

作用域——[[scope]]

作用域，即[[scope]]，是函数执行期上下文的集合。同时是javascript函数对象中仅供javascript引擎存取的属性

当函数执行时，会创建一个称为执行期上下文的内部对象。一个执行期上下文定义了一个函数执行时的环境。函数每次执行时对应的执行上下文都是独一无二的，所以多次调用一个函数会导致创建多个执行期上下文，当函数执行完毕，它所产生的执行期上下文被销毁。

作用域链

[[scope]]中所存储的执行期上下文对象的集合，这个集合呈链式链接，这种链接叫做作用域链

闭包

当内部函数被保存到外部时，将会产生闭包，闭包会导致原有作用域链不释放，造成内存泄露。

2、 隐式的执行this.xxx = xxx;
3、 隐式的返回this
// 如果显示的return {}, 则返回{}；如果return的不是对象，则返回的还是this

原型

原型是function对象的一个属性，它定义了构造函数制造出的对象的公共祖先。通过构造函数产生的对象，可以继承该原型的属性和方法，原型也是对象

  • 可以通过隐式属性proto查看原型
  • 可以通过 obj.constructor 查看对象的构造函数

原型链

原型链的连接点就是proto
object.prototype是所有对象的最终原型。即原型链的终端

undefined和null不能调用toString()，因为它们没有原型，不是对象，不能隐式转换成object，所以没有toString()
